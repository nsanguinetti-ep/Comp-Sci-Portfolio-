Computer Hardware

Computers are very complicated pieces of machinery and part of learning how to code is learning how a computer runs. There are many different elements of the internal structure of a computer that will be covered in this section. The table below depicts some of the main elements of a computer.
Parts of website Explanation
Motherboard The motherboard is the largest part of a computer and is responsible for a-lot. The motherboard can generally be found in the back of cumputers and under the keyboard of laptops. The motherboard is responsible for mangaging all the computers subsystems like the cpu the ram etc.
CPU Cpu stands for central processing unit. The CPU's main responsibility is to move memory and to process all the actions that the user is making on a computer.
RAM RAM stands for random acess memory, and is responsible for storing, saving, and using memory in a computer.
Hard Drive The hard drive is a device that transfers and stores information, Hard Drives can be used to transfer data on one computer to anothers. THe hard drive usually reads information through spinning discs
Keyboard, mouse, and monitor You most likely already know what these are,(the monitor being the computer screen) but you most likely didn't know exactly how they comunicate with each other. the mouse depicts where you will click on the monitor,and the keeyboard is responsible for what you type on the monitor, this is possible due to the CPU reading your actions and processing them.
